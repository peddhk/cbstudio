Use o addon CB Studio's no Bedrock para gravar uma cena cinemática.

03

Digite sua key aqui no site e faça download do JSON do seu replay.

Sobre o projeto

CB Studio's é uma plataforma de nuvem para replays do Minecraft Bedrock. O addon grava cada frame da câmera, mundo e configurações de câmera, e envia o JSON para esta API. Você recebe uma key e pode acessar seu replay de qualquer dispositivo.
